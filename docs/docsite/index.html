Multimedia Operability

Apply a library of 15+ classical facial manipulations to still images and movie stimuli (masking, occlusion, blurring, point-light display etc.)

Layer Manipulations

Multiple facial manipulations can be layered/composed into a single stimuli (i.e. landmark shuffling + desaturation)

Temporal Manipulation

Timing functions allow precise control over modulating the application of manipulation layers, as well as temporal shuffling of video frames
